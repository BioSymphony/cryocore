# Prompt: Public Map/Model Dossier

Create or review a map/model dossier plan for a released EMDB/PDB pair. Use
public accessions, declare the data tier, and produce an evidence packet shape
with figures, provenance, validation commands, and a claim ledger. Do not
download raw movies, particle stacks, private structures, model weights, or
license-gated tools.

Return:

- declared inputs and accession URLs
- expected artifacts and schemas
- validation commands
- claim ledger entries with claim levels
- non-claims and follow-up evidence needed for publication-grade conclusions
