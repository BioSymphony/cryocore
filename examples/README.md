# Examples

Examples are tiny and public-safe.

| Example | Purpose |
| --- | --- |
| [agent-tasks](agent-tasks/) | Copyable public-safe prompts for agents. Five prompts covering safety review, dossier planning, cloud-provider prep, wave planning, and goal-to-campaign. |
| [sample-goal-brief](sample-goal-brief/) | Worked example of `templates/goal-brief.md` filled out for the Pol Theta dossier mission. |
| [sample-linear-wave](sample-linear-wave/) | Three worked Linear-style issue bodies that form the first wave of the Pol Theta dossier mission. |
| [t2r14-open-dossier-preview](t2r14-open-dossier-preview/) | Hand-sized sample output shape for the local T2R14 demo. |
| [public-accession-metadata](public-accession-metadata/) | Metadata-only EMPIAR/EMDB/PDB accession ledger. |
| [no-download-smoke](no-download-smoke/) | No-download manifest shape for local contract validation. |
| [empiar-10204-v0](empiar-10204-v0/) | Compact campaign manifest example using public identifiers only. |

The examples are kept tiny on purpose. Downloaded maps, raw movies, particle
stacks, private structures, model weights, license files, and heavy generated
outputs live outside this directory and outside git.
