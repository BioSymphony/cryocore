# Sample Goal Brief

A worked `templates/goal-brief.md` for the Pol Theta map and model dossier
mission. Use it as a reference when your agent is filling out a brief from a
broad goal.

The brief sits in `goal-brief.md` next to this README. The matching prompt
fixture is
[examples/agent-tasks/goal-to-campaign.prompt.md](../agent-tasks/goal-to-campaign.prompt.md).

The mission targets EMD-43816 and PDB `9ASJ` (human DNA polymerase theta
helicase with AMP-PNP), the same accessions the
[Pol Theta demo](../../demos/poltheta-map-model-dossier/) uses.

## How to read it

Each section of the brief is filled out at the level a capable agent should
produce on a first pass. It names the public accessions, the resource mode,
the first artifact, the boundaries, the operator gates, and the validation
commands.

Agents reading this fixture should compare their own first draft against it
and reach for the missing fields before doing other work.
