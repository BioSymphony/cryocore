# T2R14 Open Dossier Preview

This fixture shows the shape of the tiny public-coordinate demo without
committing downloaded coordinates or generated runtime output.

Run the real local demo:

```bash
make demo-local
```

Expected runtime files appear under `.runtime/t2r14-open-dossier/`.

Included fixture files:

- `dossier_manifest.sample.json`
- `validation-summary.sample.json`
- `claim_ledger.sample.md`

These are hand-sized examples for agents and documentation. Run `make demo-local`
when you want the current artifacts on disk.
