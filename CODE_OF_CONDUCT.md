# Code Of Conduct

BioSymphony CryoCore is a scientific software project. Discussion should stay
professional, evidence-centered, and safe for public collaboration.

Expected behavior:

- Be respectful and specific.
- Separate scientific uncertainty from personal criticism.
- Keep claims tied to evidence and caveats.
- Do not post private data, credentials, provider logs, license files, or
  unpublished biological material.
- Move security or privacy reports to the private reporting path in
  `SECURITY.md`.

Maintainers may edit, hide, or remove public content that exposes private data,
credentials, unsafe instructions, harassment, spam, or unsupported biological or
medical claims.
