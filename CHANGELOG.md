# Changelog

## 0.1.0-prealpha

Initial public-staging release:

- public-safe cryo-EM control-plane docs, manifests, schemas, and validators
- metadata-only and deposited-public-artifact demo scaffolds
- provider launch, stage-progress, closeout, cost, cleanup, and claim-ledger gates
- public snapshot and release-report checks for secrets, heavy artifacts, private
  markers, and release-file completeness
- GitHub issue templates, PR checklist, CI, Dependabot, citation, notice, and
  contribution guidance

No raw cryo-EM data, private structures, provider credentials, model weights,
license files, or private history are included.
