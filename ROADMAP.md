# Roadmap

## Public v0.1

- Local-first release gate with public snapshot, schema, RunPod contract, and skill-pack validation.
- Public-safe demo dossiers for deposited structures.
- Clear claim ladder, non-claims, privacy policy, and license posture.

## Public v0.2

- More tiny public accession fixtures with cached metadata summaries.
- Additional provider profile examples for local workstation, SSH/HPC, AWS Batch, and neocloud-style pods.
- Richer figure dossier templates for Mol*, Blender, ChimeraX-gated, and static SVG routes.
- More CI examples for consumers copying the skill pack into their own repos.

## Later

- Optional packaged skill release artifacts.
- More formal provenance bundle exporters.
- Community-contributed public accession recipes.
- Pluggable closeout adapters for provider billing, artifact fetch, and cleanup proof.

All roadmap items remain bounded by the same public rules: no secrets, no
private or unpublished biological data, no raw/heavy artifacts in git, and no
license-gated execution without operator gates.
