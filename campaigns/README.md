# Campaigns

CryoCore campaigns should be evidence-first and artifact-bound.

Each campaign should declare:

- dataset accession or operator-provided data pointer
- exact inputs and subset policy
- expected artifacts
- validation commands
- compute provider and budget posture
- license-gated tools
- claim levels
- downstream handoff to CryoCore, if any

Use `campaigns/cryoem-raw-to-atomic-dossier/` as the first campaign family.
