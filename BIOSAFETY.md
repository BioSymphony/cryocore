# Biosafety

BioSymphony CryoCore is a planning, evidence, and validation-control repository. It does not perform biosafety review and does not authorize wet-lab execution.

## Scope

Public examples use synthetic fixtures or public accessions. They must not contain private structures, unpublished sequences, patient data, raw cryo-EM movies, maps, half-maps, particle stacks, model weights, or operational wet-lab protocols.

## Operator Responsibilities

Before real work moves beyond planning, operators should use the appropriate institutional review paths for:

- biological sample handling
- recombinant DNA or pathogen-related work
- collaborator, customer, or unpublished material
- dual-use concerns
- regulated, clinical, or therapeutic interpretation

## Repo Boundary

Do not use this public repo to store biological materials, raw experimental data, provider credentials, private run logs, or execution instructions that bypass local safety and license review.
